# Miami Budokan

The website of Miami Budokan, South Florida's home of traditional Japanese
swordsmanship. A not-for-profit dojo at the Japan Cultural Center in
Coral Terrace, Miami.

Pure static HTML/CSS/JS. No build tools, no CMS.

- `index.html` is the one-page site
- `blog/` holds the journal posts as standalone pages
- `assets/` holds shared styles, scripts, fonts, and images
- `hero.mp4` is the hero background film

Served via GitHub Pages.
